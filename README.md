# 🌐 Spring – Modern Responsive Landing Page

A visually appealing and fully responsive landing page designed for **digital agencies, startups, and SaaS products**. This project focuses on **clean UI/UX, structured layout, and scalability**, making it ideal for real-world frontend practice.

---
## 🚀 Live Preview: [Click Here](https://amitchandradas2004.github.io/Agency-Portfolio-landing-page-1/)
---

## 🚀 Overview

**Spring** is a modern landing page template built using **HTML5 and Tailwind CSS**, emphasizing:

* Minimalist design principles
* Clear content hierarchy
* Reusable UI sections
* Mobile-first responsiveness

This project demonstrates how to build a **real-world landing page layout** with multiple sections like services, pricing, FAQ, and call-to-action.

## 📸 Sections Breakdown

### 1. 🏠 Hero Section

* Strong headline + subtext
* Call-to-action button
* Hero image with modern layout

### 2. 💼 Services Section

* Multiple service cards
* Icons + short descriptions
* "Learn More" CTA

### 3. 👋 Welcome / About Section

* Introductory content
* Image + text alignment

### 4. 🚀 Features Highlight

* Alternating layout (image ↔ text)
* Focus on user benefits

### 5. 💰 Pricing Section

* Two pricing plans
* Feature comparison
* Clean card UI

### 6. ❓ FAQ Section

* Structured questions list
* Expandable layout (UI-ready)

### 7. 📣 Call-To-Action (CTA)

* Final conversion-focused section
* Encourages user action

### 8. 🔗 Footer

* Navigation links
* Product, company, and social links
